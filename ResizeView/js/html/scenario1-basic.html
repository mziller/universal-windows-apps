Resizing the view

Click the button to change the size of the view. For this to succeed, the current windowing model of the system must allow views to be resized freely, and the requested size must be within the range of view sizes permitted by the system.

The view size is currently × .

An app can request that the system not allow the user to resize the view below a certain size. For this to succeed, the windowing model of the operating system must allow views to be resized freely. If the requested size is outside the range permitted by the operating system, the nearest permissible values are used.
