This sample demonstrates ...

Specifically, this sample covers:

-   Enumerating Radios on the device
-   Registering for state changes on individual Radios
-   Toggling Radio state changes

System requirements
-----------------------------

Windows 10 Technical Preview

Build the sample
----------------

1.  Start Visual Studio 2015 and select **File** \> **Open** \> **Project/Solution**.
2.  Go to the directory to which you unzipped the sample. Go to the directory named for the sample, and double-click the Visual Studio 2015 Solution (.sln) file.
3.  Press Ctrl+Shift+B, or use **Build** \> **Build Solution**

Run the sample
----------------
The scenario has buttons you can use to perform the actions described in the scenario text. If you do not have any radios on your system, you will not
be able to experience the scenario.
